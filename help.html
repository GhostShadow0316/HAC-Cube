How to use the Commands?


command description
cube [cube] change cube
next next scramble
punish [+2 / DNF] set punish to current solve
leave empty for clear punish
view [index] view solve to no. [index]
edit punish [index] [+2 / DNF] set punish to solve no. [index]
leave empty for clear punish
remove [idx] remove solve no. [index]
copy { scramble / sc } copy current scramble
copy time copy current solve time
session [name]
session { change / = } [name]		 switch between sessions
session { add / + } [name] add new session
session { remove / - } [name] remove existed session
export export sessions as a json file
import import sessions from a json file
