How to use the Commands?


command description
cube [cube] change cube cube 3
next next scramble next
punish [+2 / DNF] set punish to current solve
leave empty for clear punish		 punish dnf
view [index] view solve to no. [index] view 13
edit punish [index] [+2 / DNF] set punish to solve no. [index]
leave empty to clear punish		 edit punish 13 dnf
remove [idx] remove solve no. [idx] (support negative index) remove 13
remove [idx1]~[idx2] remove solve from [idx1] to [idx2] remove 7~13
copy { scramble / sc } copy current scramble copy sc
copy { time / tm } copy current solve time copy time
session { change / = } [name] switch between sessions session = default
session { add / + } [name] add new session
session { remove / - } [name] remove existed session
export export sessions as a json file
import import sessions from a json file
restore restore sessions from the auto backup last import

version show version
reload
refresh		 refresh page
reset reset page
help show this page
